# Xeon Deal Assistant

Next.js app ready for GitHub and Vercel.

## Local run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy on Vercel

1. Create a GitHub repository.
2. Upload/push all files from this folder.
3. Go to Vercel > Add New Project.
4. Import the GitHub repository.
5. Keep default settings and click Deploy.
