# Xeon Deal Assistant

Vercel-ready Next.js app with Lenovo + Intel branding, dark dashboard UI, active filters, radar chart, and Excel export.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Create a GitHub repository.
2. Upload this full folder to the repository.
3. Import the repository in Vercel.
4. Keep default settings and deploy.
