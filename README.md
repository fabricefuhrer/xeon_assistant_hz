# Xeon Deal Assistant

Clean Vercel-ready Next.js version.

## Deploy
1. Upload the contents of this folder to the GitHub repo root.
2. Confirm the root contains `app`, `public`, `package.json`, `tsconfig.json`, and `next.config.js`.
3. In Vercel, redeploy with **Clear Build Cache**.

## Local test
```bash
npm install
npm run build
npm run dev
```
