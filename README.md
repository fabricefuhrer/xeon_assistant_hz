# Xeon Deal Assistant v2

Clean split-component version of the Lenovo/Intel Xeon Deal Assistant.

## Rules included
- ST50 V3 and SR250 V3: Intel Pentium, Xeon E, Xeon 6300 only.
- SR630 V4: Xeon 6500/6700 P-Cores and Xeon 6700 E-Cores.
- SR650 V4, SR680a V4, SR850 V4, SR860 V4: Xeon 6500/6700 P-Cores only.
- No fallback recommendation to incompatible CPUs.
- Key Metrics calculated only from the filtered compatible pool.
- Commercial Alternatives added without changing the overall black dashboard UI.

## Deploy
Upload contents to GitHub root, then deploy on Vercel.
